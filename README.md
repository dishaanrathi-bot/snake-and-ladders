# Snake and Ladder Game in c

## Project Overview

The Snake and Ladder Game is a simple board game project developed to demonstrate basic programming concepts such as loops, conditions, functions, and random number generation. The game follows the traditional rules of Snake and Ladder where players roll a dice and move forward while encountering snakes and ladders.

## Objectives

1. To understand basic game logic
2. To learn random number generation
3. To practice conditional statements
4. To simulate a turn-based game
